# VPC MT-50CM3 Right Stick Template

This directory contains visual templates for the VPC (VirPil Controls) MT-50CM3 right-handed stick.

## Required Files

- **Device image**: Base image of the device (e.g., `vpc_mt50cm3_right.png` or `.jpg`)
- **Template image**: Annotated version showing button/axis locations
- **Overlay SVG**: SVG overlay file for mapping controls onto the device image

## Device Information

- **Full Device Name**: RIGHT VPC Stick MT-50CM3
- **Product ID**: `{43903344-0000-0000-0000-504944564944}`
- **Type**: Joystick
- **Instance**: Typically instance 1 in Star Citizen profiles

## Match Patterns

When adding to `template_registry.json`, use patterns like:
- "RIGHT VPC Stick MT-50CM3"
- "VPC MT-50CM3"
- "VirPil MT-50CM3 Right"
