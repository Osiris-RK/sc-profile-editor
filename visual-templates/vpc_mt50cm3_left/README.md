# VPC MT-50CM3 Left Stick Template

This directory contains visual templates for the VPC (VirPil Controls) MT-50CM3 left-handed stick.

## Required Files

- **Device image**: Base image of the device (e.g., `vpc_mt50cm3_left.png` or `.jpg`)
- **Template image**: Annotated version showing button/axis locations
- **Overlay SVG**: SVG overlay file for mapping controls onto the device image

## Device Information

- **Full Device Name**: LEFT VPC Stick MT-50CM3
- **Product ID**: `{83903344-0000-0000-0000-504944564944}`
- **Type**: Joystick
- **Instance**: Typically instance 2 or 3 in Star Citizen profiles

## Match Patterns

When adding to `template_registry.json`, use patterns like:
- "LEFT VPC Stick MT-50CM3"
- "VPC MT-50CM3"
- "VirPil MT-50CM3 Left"
