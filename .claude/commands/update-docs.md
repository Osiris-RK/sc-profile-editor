This command conducts updates all the docs in the docs directory to the current state of the application

# Invocation

To invoke, type `/update-docs`

# Instructions

Perform the following steps for a release:

1. Review all the docs in the `docs` directory.
2. If it hasn't been run in this session yet, run `/get-context` to understand the current state of the application
3. Update the documentation in the docs directory to reflect any changes that have happened since the last time the docs were update.
