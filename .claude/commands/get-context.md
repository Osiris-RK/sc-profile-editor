This command instructs claude to get a fuller breadth of context than it does by default.


# Invocation

This command is invoked with `/get-context`

# Instructions

When invoked, perform the following steps:

1. Read all files in the `.claude/logs` directory.
2Read all documentation in the `docs` directory. 
3. Study all files ending in .xml, .pdf, .json and .py.
4. Look at all commits since claude last ran so you can see what was changed recently
